简介

.NET 是由微软开发的自由和开源的计算机软件框架，支持多种语言编写应用程序。.NET 提供了语言互操作性，每种语言都可以使用其他语言编写的代码。

使用方法

.NET 可用于开发各种类型的应用程序，包括但不限于：

  • Web 应用程序
  • 移动应用程序
  • 桌面应用程序
  • 云服务
  • 游戏开发
  • IoT 应用程序

开始使用

要开始使用 .NET，请先安装 .NET SDK，然后创建您的第一个项目。
